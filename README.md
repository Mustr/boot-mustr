### **Boot Mustr**

基于spring boot 2.0.4开发的项目



#### 相关技术

------

1、后端

- 核心框架：Spring Boot
- 安全框架：spring security
- 模板引擎：Thymeleaf
- 持久层框架：MyBatis
- 数据库连接池：Alibaba Druid
- 日志管理：SLF4J
- 工具类：Apache Commons、Jackson
- 通信：spring-websocket
- 文件存储：Minio



2、前端

- JS框架：jQuery
- 客户端验证：JQuery Validation
- 页面框架：xenon
- 弹窗，数据表格等：layui，bootstrap
- 通信：sockJS，stomp, webscoket
